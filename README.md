安装
----
- **在线安装:**
  - `File` -> `Setting` -> `Plugins` -> 搜索 `Doc View`

- **手动安装:**
  - [下载插件](https://github.com/liuzhihang/doc-view/releases) -> `File` -> `Setting` -> `Plugins` -> `Install Plugin from Disk...`

使用
----
- 右键菜单选择 `Doc View`
