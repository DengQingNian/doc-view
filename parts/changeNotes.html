  • 1.1.7
    1. Doc Editor UI 调整为树列表
    2. Doc Editor 支持 Swagger 注解的必填和描述
    3. 修改 Dubbo 接口无法上传 YApi 的情况
    4. 修复 String 类型字段解析错误

    6. 支持解析字段行注释,并在编辑后转换为 Java Doc 注释
    7. 其他一些问题的修复
    8. 备注:当前版本使用 swingx-core 依赖生成 TreeTable, 导致插件包占用空间变大, 需要后期调整, 具体可参考: https://youtrack.jetbrains.com/issue/IDEA-275249 ,如果小伙伴有其他解决方式, 希望可以联系我,或者提 Issue.
